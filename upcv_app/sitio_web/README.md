# sitio_web

Frontend público corporativo y modelos de contenido para Tecnologías de Guatemala.

## Administración de contenido
Toda la gestión se realiza en el **CMS privado** con vistas propias:
- `/cms/` dashboard
- `/cms/configuracion/`
- `/cms/heroes/`
- `/cms/paginas/`
- `/cms/servicios/`
- `/cms/proyectos/`
- `/cms/proyectos/imagenes/`
- `/cms/testimonios/`
- `/cms/aliados/`
- `/cms/faqs/`
- `/cms/bloques/`
- `/cms/mensajes/`
